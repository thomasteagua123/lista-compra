1-Instalar Bun en la computadora
2-Clonar o descargar el proyecto desde GitHub
3-Abrir la terminal en la carpeta del proyecto
4-Ejecutar:
bunx create-expo-app lista-compras
cd lista-compras
bun start
5-Esperar a que se abra Expo con el código QR
6-Conectar el celular y la computadora al mismo WiFi
7-Instalar la app Expo Go en el celular
8-Abrir Expo Go
9-Escanear el código QR
10-Escribir un producto en el campo de texto
11-Presionar el botón "Agregar"
12-Ver el producto en la lista
13-Tocar un producto para marcarlo como completado
14-Mantener presionado un producto para eliminarlo
15-Si no hay productos, aparecerá un mensaje indicando que la lista está vacía
